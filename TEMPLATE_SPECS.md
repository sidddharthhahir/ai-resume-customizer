# Custom Template Design Specifications

## Template Name: Professional Sidebar

### Layout Structure
- **Two-column layout**: Left sidebar with profile photo and contact info, right main content area
- **Profile Photo**: Circular image (150x150px) in top-left corner
- **Header Section**: Name prominently displayed, contact information below photo

### Header/Contact Section
- Name: Large bold font (24pt+)
- Contact info arranged horizontally: Email, Phone, Location
- Social links: LinkedIn, Portfolio, GitHub with icons
- Work authorization info below

### Section Headers
- Icon + Section title format
- Icons used: briefcase, code, folder, graduation cap
- Black background with white text for headers
- Horizontal line separator below each header

### Content Sections

#### Professional Summary
- Paragraph text describing role and experience
- Focus on technologies and key achievements

#### Technical Skills
- Organized by category (Frontend, Backend, Databases, AI/ML, DevOps & Tools, Cloud)
- Each category in a bordered box
- Two columns for some categories

#### AI-Accelerated Projects
- Project title with technologies in italics
- Date range on the right
- Bullet points describing achievements
- Links to Live demos and GitHub

#### Projects
- Similar structure to AI-Accelerated Projects
- Project name with technologies
- Date range
- Bullet points with key features and metrics
- GitHub links

#### Education
- Degree and institution name
- Date range on right
- Location below

### Styling Details
- **Font**: Clean sans-serif (similar to Arial/Helvetica)
- **Colors**: Black text on white background, black headers
- **Icons**: Monochrome icons for sections and links
- **Spacing**: Generous margins and padding
- **Photo**: Circular with subtle shadow/border
- **Borders**: Thin horizontal lines for section separation

### ATS Optimization
- Clean semantic structure
- No complex graphics or tables
- Text-based layout
- Proper heading hierarchy
- Standard fonts for maximum compatibility
